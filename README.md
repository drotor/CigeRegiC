# CigeRegiC (formerly NicReg) Manage and Regulate Cigarette Inventory  
  
## To clean the registry:  
  
###  A. On this local repository  
   1. Type `cd ~/Projects/<...>/CigeRegiC/`  
   2. Type `ungit`  
      a. Squash back branch **main** to initial node  
      b. Select **Amend last commit**  
      c. Commit and force push
###  B. On other local repositories  
  1. Run `update-NicReg.sh`  
#### or  
  
   1. Type `cd ~/Projects/<...>/CigeRegic/`  
   2. Type `git fetch`  
   3. Type `git rebase`
